# TKT-025: MCP update must enforce state transitions

## Description
`internal/mcp/handlers.go` updates ticket state without calling `ticket.ValidateTransition`,
so MCP clients can bypass the CLI state machine and write invalid workflow jumps.

## Acceptance Criteria
- [ ] MCP update rejects invalid state transitions the same way `docket update` does
