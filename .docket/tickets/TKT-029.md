# TKT-029: Suppress Cobra usage text on expected quality-gate failures

## Description
Commands like `docket ac check` and `docket check` print their human-readable failure
output and then Cobra appends usage text because they return sentinel errors. That
pollutes hook and CI output.

## Acceptance Criteria
- [x] Expected failing quality-gate commands exit non-zero without printing command usage : Implemented and verified by tests in this change set.
