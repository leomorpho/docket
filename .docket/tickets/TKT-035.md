# TKT-035: Add semantic provider interface

## Description
Why: provider-dependent logic should not leak into indexing or ranking code.

Scope:
- Add a Go interface for local embedding providers.
- Define explicit request and response types for status checks and embedding batches.

Files to touch:

Non-goals:
- No provider registry or real embeddings yet.

## Acceptance Criteria
