# TKT-142: Core: Implement cross-worktree claiming in shared .git directory

## Description
Implement the underlying storage and logic for "Ticket Claiming" using the shared Git metadata directory. This provides the coordination backbone for parallel agents.

## Acceptance Criteria
- [ ] Implement `GetGitCommonDir()` to find the absolute path of the shared `.git` folder.
- [ ] Implement `internal/claim` package with:
    - `Claim(ticketID, worktreePath, agentID)`: Creates a claim JSON in `$(GIT_COMMON)/docket/claims/`.
    - `Release(ticketID)`: Removes the claim.
    - `GetClaim(ticketID)`: Returns claim metadata.
- [ ] Command line: `docket claim <ID>` and `docket release <ID>` (hidden by default).
- [ ] Data structure: `{"agent_id": "...", "worktree": "...", "at": "..."}`.
