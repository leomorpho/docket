# TKT-144: Implement `docket install --skill` for Gemini CLI integration

## Description
Create a command to automatically generate and install a Docket "Skill" for the Gemini CLI. A Skill provides specialized domain knowledge and tool-use instructions directly to the agent. This will force agents to use Docket's MCP tools correctly and prevent them from attempting to modify the raw ticket files directly.

## Acceptance Criteria
- [ ] Implement `docket install --skill` command.
- [ ] Command should detect the presence of `gemini-cli` or default to `~/.gemini/skills/docket/`.
- [ ] Automatically generate a `SKILL.md` file with the following content:
    - MCP server tool descriptions.
    - Explicit directive: "CRITICAL: Do not edit .docket/tickets/*.md directly. Use the 'docket' MCP tools for all modifications."
    - Instructions on ticket claiming (TKT-142/143).
    - Guidelines on using temporary files for large payloads (TKT-146).
- [ ] Provide clear feedback on whether the skill was installed or if manual steps are needed.
