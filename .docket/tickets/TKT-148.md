# TKT-148: Implement `docket fix` to repair ticket signatures and teach agents

## Description
Create the "Tutor" command, `docket fix`, which automates the repair of ticket files that have been directly edited by an LLM or human. Instead of just failing, Docket will now intelligently detect the intended change, apply it through the official storage logic (generating a valid `write_hash`), and then explicitly show the LLM the CLI command it *should* have used.
