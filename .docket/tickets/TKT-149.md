# TKT-149: Enhance pre-commit validation with 'Tutor' feedback

## Description
Integrate the `docket fix` "Tutor" logic into the pre-commit validation flow. This ensures that when an agent makes a mistake and tries to commit a manually edited ticket, Docket doesn't just block them but also provides a clear, actionable path to correction and learning.

- [x] Update `lefthook.yml` (pre-commit) to suggest `docket fix --all`.
