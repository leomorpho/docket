# TKT-152: Deprecate and hide the TUI Kanban board

<!-- docket: Direct edits are allowed. After editing this ticket, run `docket validate TKT-152` before committing. -->

## Description
The `docket board` TUI (based on Bubbletea) is being deprecated in favor of better CLI listings and the upcoming web-based UI. We need to signal this deprecation to users and prevent LLMs from attempting to use the command.

## Acceptance Criteria
- [ ] Add a deprecation warning to the `docket board` command that prints to stderr.
- [ ] Remove the `board` command from the `help-json` machine-readable manifest (internal/mcp/ or cmd/helpjson.go).
- [ ] Update `cmd/board.go` and `root.go` documentation to mark it as deprecated.
- [ ] Point users toward `docket list` or the future web UI.
