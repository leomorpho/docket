# TKT-171: Implement docket start: Autonomous Agentic Workflow

## Description
Create a docket start command that serves as an entry point for autonomous agentic work.
- Automatically selects the next best ticket (based on priority, blockers, and state).
- Prompts the agentic environment to pick up the ticket and work on it.
- Loops through eligible tickets until all possible work is complete.

Acceptance Criteria:
- [ ] docket start correctly identifies the next unblocked high-priority ticket.
- [ ] The command provides a clear instruction/prompt for the agent to begin work.
- [ ] Support for auto-mode where it continues to the next ticket after completion.
