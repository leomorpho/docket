## Description
Add explicit runtime identity for agent-managed work.

Context:
- Later gates such as commit linkage context reset and secure review need a durable record of what run is active and what workflow hash it was pinned to.
- Prefer a small explicit manifest over implicit assumptions spread across multiple packages.

What to do:
- Define the manifest format and storage location.
- Record it when a Docket-managed run starts.
