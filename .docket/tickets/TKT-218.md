## Description
Context: Agents and automation need a dedicated command for full capabilities without bloating start output.

Objective: Implement docket capabilities command that renders canonical contract as markdown and JSON.

Scope:
- Markdown output for human-readable capability reference.
- JSON output for machine consumption.
- Include active adapter metadata plus contract version and hash.

Out of scope: Interactive setup flows.

Delivery notes: Capabilities output must come from canonical contract to avoid drift.
