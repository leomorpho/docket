## Description
Context: Each target adapter must prove bootstrap, start, doctor, and capabilities interoperability.

Objective: Add matrix-style integration coverage for the three supported adapters in simulated environments.

Scope:
- Per-adapter bootstrap artifact checks.
- Per-adapter start readiness strip checks.
- Per-adapter doctor pass and fail transitions.
- Capabilities output adapter metadata verification.

Delivery notes: Matrix tests should be deterministic and runnable in local CI.
