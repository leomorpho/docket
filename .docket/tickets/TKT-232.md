# TKT-232: Add global automation mode for non-interactive deterministic CLI behavior

## Description
Context: Bots need a deterministic mode that disables prompts and background noise without bespoke per-command flags.

Scope:
- Global flag and env variable for automation mode.

- Consistent output formatting guarantees for machine consumers.

Out of scope: Platform-specific adapter logic.
