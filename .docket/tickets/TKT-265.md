# TKT-265: Add CLI commands to inspect and search stored learnings

## Description
Turn learnings into a first-class product surface by adding CLI commands to list, inspect, and search stored learning rules. The commands should make the current learning store visible and trustworthy and be implemented with tests first.

## Acceptance Criteria
- [ ] The CLI exposes commands to list and search stored learning rules with enough detail to show category, rule text, and source.
- [ ] Automated tests cover empty-state behavior, search filtering, and stable human or JSON output for stored learnings.
