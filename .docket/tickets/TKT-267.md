# TKT-267: Add an explicit learn-rule capture command

## Description
Add a first-class command for intentionally recording a learning rule instead of relying only on passive LEARN: ingestion from comments or session files. The command should validate input, write through the existing learning store, and be test-driven.

## Acceptance Criteria
- [ ] The CLI exposes a learn-rule capture command that records a categorized learning rule with source metadata in the existing learning store.
- [ ] Automated tests cover successful capture, deduplication behavior, and clear errors for invalid or incomplete learning input.
