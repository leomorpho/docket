# TKT-270: Add a context-optimizer command for compact LLM task briefs

## Description
Add a command that assembles a compact, token-conscious task brief from ticket context, related work, learnings, and recent activity. The output should help agents start work quickly without reading every raw surface and should be built test-first.

## Acceptance Criteria
- [ ] The CLI exposes a context-optimizer command that produces a concise task brief for a ticket using existing Docket context sources.
- [ ] Automated tests verify that the optimized brief includes the required context sources while remaining bounded and predictable in shape.
