# TKT-278: Add bulk roadmap-to-tickets authoring flow for one-line worklists

## Description
Improve Docket authoring for roadmap grooming by adding a flow that can turn a concise worklist into multiple draft tickets without repetitive shell loops. The flow should preserve room for grooming details, make parent assignment straightforward, and be implemented from failing tests.

## Acceptance Criteria
- [ ] Docket provides a bulk authoring flow that can ingest a concise worklist and create multiple draft tickets with stable titles and parent assignment.
- [ ] Automated tests cover bulk authoring from a worklist input and verify that created tickets remain editable and groomable after import.
