# TKT-280: Add first-class skill definitions and metadata for Docket capability surfaces

## Description
Replace the current placeholder-oriented skill inventory with first-class Docket skill definitions that describe real capabilities, invocation intent, triggers, and discovery metadata. The definitions should map cleanly onto the standard Agent Skills format and stay stable across CLI, MCP, and adapter installs, with tests written first.

## Acceptance Criteria
- [ ] The runtime capability contract and rendered discovery surfaces expose first-class Docket skill metadata for real capabilities in a shape that can drive standard Agent Skills artifacts.
- [ ] Unit tests and regression tests verify that skill metadata stays synchronized across the runtime contract, Agent Skills artifacts, CLI capability output, MCP discovery, and adapter-rendered artifacts.
