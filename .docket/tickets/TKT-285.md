# TKT-285: Add standard Agent Skills artifact generation for Docket capabilities

## Description
Generate standard Agent Skills artifacts for Docket capabilities so the same skill pack can be consumed across Claude Code, Codex, Gemini, Cursor, and other spec-aligned tools. Treat the Agent Skills format as the primary portable packaging layer, derive it from canonical Docket skill definitions, and implement the work from failing tests first.

## Acceptance Criteria
- [ ] Docket generates standard Agent Skills artifacts from canonical skill definitions using a stable directory layout and metadata shape that supports cross-agent portability.
- [ ] Unit tests and regression tests verify Agent Skills artifact generation, schema stability, and alignment with the canonical Docket skill definitions and capability snapshot.
