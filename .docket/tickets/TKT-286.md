# TKT-286: Add unified Agent Skills installer for project-local and user-global directories

## Description
Add a unified installer that detects supported environments and installs Docket Agent Skills into project-local and user-global directories with clear reporting. The installer should hide directory fragmentation from users, prefer a canonical Docket-managed skill source, and be built from failing tests first.

## Acceptance Criteria
- [ ] Docket installs canonical Agent Skills into supported project-local and user-global directories with clear reporting about what was installed and where.
- [ ] Unit tests and regression tests cover environment detection, install path selection, idempotent reinstall behavior, and clean fallback messaging when no supported skill directories are available.
