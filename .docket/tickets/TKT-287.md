# TKT-287: Add canonical Docket skill pack and starter bundle for supported agents

<!-- docket: Direct edits are allowed. After editing this ticket, run `docket validate TKT-287` before committing. -->

## Description
Package the highest-value Docket skills into a small recommended starter bundle that can be installed consistently across supported agents. Keep the bundle opinionated, aligned with the canonical skill definitions, and tested first so Docket ships a known-good baseline rather than a loose collection of skills.

## Acceptance Criteria
- [ ] Docket ships a canonical starter skill bundle that includes the supported high-value skills and can be rendered or installed consistently across supported agents.
- [ ] Unit tests and regression tests verify the starter bundle contents, metadata stability, and install compatibility across the supported adapter matrix.
