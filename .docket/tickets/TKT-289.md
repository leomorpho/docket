# TKT-289: Add cross-agent skill sync from canonical Docket skill directory

<!-- docket: Direct edits are allowed. After editing this ticket, run `docket validate TKT-289` before committing. -->

## Description
Manage a canonical Docket skill directory and add sync logic that fans its contents out to adapter-specific skill locations as needed. This should reduce duplication, keep installs consistent across agents, and be implemented from failing tests first.

## Acceptance Criteria
- [ ] Docket maintains a canonical skill directory and can sync or mirror it into adapter-specific skill locations without changing the canonical contents.
- [ ] Unit tests and regression tests cover canonical skill sync, drift detection, idempotent updates, and adapter-specific sync behavior across supported environments.
